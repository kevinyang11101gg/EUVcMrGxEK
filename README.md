# 项目简介

## 核心功能
- 用户系统：注册登录、JWT认证、权限管理
- 数据概览：KPI指标、趋势图、排名、同比环比
- 数据大屏：全屏展示、实时刷新、多图联动
- 趋势分析：月度、季度、日趋势、同比环比计算
- 区域分析：地图热力图、省份联动、区域排名
- 产品分析：BCG矩阵、品类分析、产品排名
- 客户分析：RFM模型、客户分层、消费行为
- 数据导入：Excel导入、模板下载、数据校验

## 适用场景
- 企业内部销售数据分析
- 销售团队绩效评估
- 市场营销策略优化
- 客户关系管理

## 技术特点
- 前端：Vue 3 + Element Plus + ECharts + Vite，提供流畅的用户界面和图表展示
- 后端：Python Flask + JWT + SQLAlchemy，确保数据安全和高效的处理
- 数据库：MySQL 8.0，支持高并发和大规模数据存储
- 架构：前后端分离 + RESTful API，便于扩展和维护
